1. 最佳推薦：FinMind（最方便、資料完整）
FinMind 是專為台股設計的開源金融資料平台，提供 75+ 種資料集，包含每日股價（含成交量）、週轉率相關計算基礎等，每天更新。

關鍵資料集：
TaiwanStockPrice 或 taiwan_stock_daily：包含個股每日 成交量 (volume)、成交值、股本等。
可一次取得多檔或全部股票的歷史/當日資料。
週轉率 = 成交量 / 流通股數（或發行股數），可從 TaiwanStockInfo（股本/流通資訊） + 成交量資料自行計算，或用市值/股本相關資料集輔助。

使用方式（Python）：Pythonfrom FinMind.data import DataLoader

2. TWSE 官方 OpenAPI（強烈推薦，穩定無被擋風險）
TWSE 提供官方 OpenAPI（https://openapi.twse.com.tw/），直接支援多種市場報告。

重點端點：
/exchangeReport/TWT53U：集中市場每日成交量前二十名證券（可直接拿 top 20）。
其他如每日成交資訊、統計報表等。
可取得全市場資料後自行排序 top 100。

結合 TWSE 歷史交易資料頁面或 CSV 下載，也能補足。
TPEX（櫃買） 也有類似 OpenAPI，提供成交量/值/週轉率排行。

使用：直接用 requests GET 這些端點，JSON 格式乾淨。官方 API 有使用規範，但日常排名用途通常沒問題。

如何避免被擋（關鍵重點）

優先用 API：TWSE OpenAPI、FinMind、data.gov.tw 開放資料 —— 這些都是官方/半官方允許的介接方式，幾乎不會被擋。
爬蟲防護措施（如果一定要用）：
加隨機延遲：每次 request 間隔 1~5 秒 + random（time.sleep(random.uniform(1,3))）。
User-Agent 輪換：模擬不同瀏覽器（用 fake-useragent 套件）。
Proxy / IP 輪換：若長期大量跑，可用住宅 Proxy（但成本高）。
只爬必要資料：不要每分鐘全掃，只在收盤後（15:30 後）跑一次每日排名。
遵守 robots.txt + 使用條款：TWSE 有明確規範，勿過度。
Header 完整：帶上 Referer、Accept 等。
分散請求：分批取得股票清單（先用 TaiwanStockInfo 拿所有 stock_id）。


實際建議：先用 FinMind 開發原型（最快），再補 TWSE OpenAPI 驗證官方資料。每天只跑一次（例如晚上排程），完全不用擔心被擋。需要 top 100 歷史排名時，用 FinMind 拉一段期間資料排序即可。
